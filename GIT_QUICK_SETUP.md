# ⚡ Git Quick Setup - Windows

## 🚀 Fastest Way

### Step 1: Download
1. Browser में जाएं: **https://git-scm.com/download/win**
2. **Download** button click करें
3. File download होने दें

### Step 2: Install
1. Downloaded file पर **double-click** करें
2. **Next → Next → Next...** (सभी default options)
3. **Install** click करें
4. Wait करें (2-5 minutes)
5. **Finish** click करें

### Step 3: Verify
1. **Command Prompt** खोलें (नया)
2. Type करें:
```bash
git --version
```
3. Version number दिखना चाहिए ✅

### Step 4: Configure (First Time)
```bash
git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"
```

### Step 5: Use Git
```bash
cd C:\Users\lenovo\Downloads\MyProjects\websevix2
git init
```

---

**Done!** अब Git ready है! 🎉
